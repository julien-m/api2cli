Context discovery

--help

The agent doesn't read a manual.
It walks a tree. Level by level. Only what it needs.

$ typefully-cli --help
Commands:
  social-sets  Manage social sets
  drafts       Manage drafts
  media        Manage media
  tags         Manage tags
  queue        Manage queue

~90 tokens. Agent sees the full API.

CLI→social-sets→action→execute
$ typefully-cli social-sets --help
Commands:
  list               List all social sets
  get <id>           Get social set details
  resolve-linkedin  Resolve LinkedIn company URL

~50 tokens. Agent drills into the resource it needs.

CLI→drafts→create→execute
$ typefully-cli drafts create --help
Options:
  --text <text>           Post content (required)
  --platform <platforms>  x,linkedin,threads,bluesky
  --publish-at <when>     "now", "next-free-slot", or ISO
  --json                 Output as JSON

~80 tokens. Exact flags. No guessing.

CLI→drafts→create→execute
$ typefully-cli drafts create abc123 \
    --text "Launching api2cli today!" \
    --platform x,linkedin \
    --publish-at next-free-slot --json
{ "ok": true, "data": { "id": "draft_7kx..." } }

No JSON to build. No endpoint to guess.
Just flags.
